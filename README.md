# ⚡ CodeSync Pro

**CodeSync Pro** is a high-performance, enterprise-grade collaborative IDE built for real-time code sharing and instant execution. Inspired by VS Code's professional architecture, it provides a seamless environment for developers to work together, anywhere in the world.

![IDE Mockup](https://raw.githubusercontent.com/himanshu-linux/CodeSync-Pro/main/client/public/logo192.png)

## 🚀 Key Features

- **Real-Time Collaboration:** Powered by Socket.IO for sub-millisecond synchronization of code changes and cursor movements.
- **Professional IDE Architecture:** Features an Activity Bar, Adaptive Sidebar (Explorer/Chat), and a Tabbed Editor system.
- **Multi-Language Execution:** Integrated with the Judge0 API for asynchronous code execution across multiple languages (Python, JavaScript, C++, etc.).
- **Collaborative Chat:** Built-in IDE chat to communicate with your team without leaving your workspace.
- **Midnight Pro Aesthetics:** A sleek, glassmorphic dark theme designed for focus and productivity.
- **Mobile Responsive:** (Coming Soon) Optimized for power users on Desktop.

## 🛠️ Technology Stack

- **Frontend:** React 18, CodeMirror (Advanced Editor Features), CSS3 (Glassmorphism).
- **Backend:** Node.js, Express, Socket.IO.
- **Compiler Engine:** Judge0 CE (Asynchronous API).
- **Design System:** Custom "Midnight Slate" CSS architecture.

## 📦 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/himanshu-linux/CodeSync-Pro.git
   cd CodeSync-Pro
   ```

2. **Setup Client:**
   ```bash
   cd client
   npm install
   npm start
   ```

3. **Setup Server:**
   ```bash
   cd server
   npm install
   npm start
   ```
